Transformer attention is a mechanism that allows the model to weigh the importance of different tokens when encoding a sequence. For each token, it computes a query, key, and value vector. The attention score between two tokens is the dot product of the query of one with the key of the other, scaled by √dk and passed through a softmax.

Attention(Q,K,V) = softmax(QKᵀ/√d_k)V

import torch
import torch.nn.functional as F

def scaled_dot_product_attention(q, k, v):
    d_k = q.size(-1)
    scores = torch.matmul(q, k.transpose(-2, -1))
    scores = scores / d_k ** 0.5
    attn = F.softmax(scores, dim=-1)
    return torch.matmul(attn, v), attn
